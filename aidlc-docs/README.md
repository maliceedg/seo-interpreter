# AI-DLC

This project uses the [AI-Driven Development Life Cycle](https://aws.amazon.com/blogs/devops/ai-driven-development-life-cycle/) methodology.

- Workflow rule: `.cursor/rules/ai-dlc-workflow.mdc` (always apply)
- Rule details: `.aidlc-rule-details/` (from [awslabs/aidlc-workflows](https://github.com/awslabs/aidlc-workflows) v1.0.1)
- Persistent artifacts: `aidlc-docs/`
- Design references: `design/stitch-v2/`

Start work by stating intent in chat. The agent follows Plan → Clarify → Approve → Implement.
